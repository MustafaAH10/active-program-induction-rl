# Asset and license notes

The Manhattan-like fixture and procedural geometry in this repository are original stylized data created for SkyFoundry. The bundled planter GLB was generated locally from SkyFoundry's own checked-in dimensional recipe; it contains no third-party source asset. No live OpenStreetMap data, third-party textures, HDR images, or proprietary building models are bundled. UI icons are supplied by the project's `lucide-react` dependency under its published license.

If optional external assets are added, preserve source, license, author, retrieval date, file hash, and transformation metadata with the project artifact.
